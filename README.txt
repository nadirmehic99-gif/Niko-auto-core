Niko Auto Core USA customer price search

Files:
- index.html = customer login/search page
- admin.html = user manager and serial image manager
- users.json = customer usernames/passwords
- images.json = serial images for customer search
- Live_Market_Price.xlsx = price list

Customer page shows ONLY:
- serial/code
- final customer price
- catalytic converter image(s)

Customer page does NOT show Pt/Pd/Rh PPM, weight, market numbers, or formula.

How to add images:
1. Open admin.html
2. Login with admin password
3. Type serial code
4. Choose 1-4 pictures
5. Click Add / Replace Serial Images
6. Click Download images.json
7. Replace images.json in GitHub with the downloaded file

GitHub Pages cannot permanently save uploaded images by itself. This app uses images.json so you can update photos safely by replacing one file.
